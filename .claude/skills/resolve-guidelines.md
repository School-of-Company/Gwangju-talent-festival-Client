# Resolve Guidelines

PR 리뷰 댓글 답변 분류 기준과 작성 방법.

## 분류 기준

| 분류 | 기준 |
|------|------|
| **반영 완료** | 이미 코드에 반영된 피드백 |
| **반영 예정** | 수정이 필요한 피드백 |
| **스킵** | 범위 밖이거나 동의하지 않는 피드백 |

## 답글 작성 방법

**반영 완료** — 어떻게 수정했는지 + 커밋 해시
```
`clearAllCookies()` 헬퍼를 추가하고 `beforeEach`에서 호출하도록 수정했습니다. (f812496)
```

**반영 예정** — 수정 방향과 이유
```
다음 커밋에서 `setLocationSearch()` 헬퍼로 추출하겠습니다.
```

**스킵** — 이유 명확히, 대안이 있으면 함께
```
해당 부분은 Next.js 빌드 산출물 수준으로 판단해 테스트 범위에서 제외했습니다.
```

## 규칙

- 모든 답글은 한국어
- 간결하고 명확하게
- 수정 참조 시 커밋 해시 포함

## 답글 게시 명령

```bash
gh api repos/:owner/:repo/pulls/comments/{comment_id}/replies \
  -X POST -f body="<한국어 답글>"
```
